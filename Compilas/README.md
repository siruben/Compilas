# COMPILAS? — O Jogo do T 🎮

Jogo estilo Tetris com tema de programação, visual neon/cyberpunk.

## Como jogar

| Tecla | Ação |
|-------|------|
| ← → | Mover |
| ↑ | Rodar |
| ↓ | Descer |
| SPACE | Hard drop |
| P | Pausa |

No telemóvel: usa os botões no ecrã.

## Peças

- 🟣 **T Pequeno** — topo + 1 bloco de haste
- 🔵 **T Médio** — topo + 2 blocos de haste
- 🟢 **T Grande** — topo + 3 blocos de haste
- 🟠 **Base T** — só o topo (3 blocos horizontais)
- 🔴 **Cubo** — 1 bloco único

## Mensagens

- ✅ **COMPILADO!** — linha(s) limpa(s)
- ❌ **FALHA CRÍTICA! STACK OVERFLOW!** — game over
