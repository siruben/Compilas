# COMPILAS? — O Jogo do T 🎮

Jogo estilo Tetris com peças T de haste variável, estilo neon/cyberpunk, otimizado para telemóvel.

## Peças
- 🟥 **Cubo** — 1 bloco
- 🟧 **Base T** — só o topo (3 blocos)
- 🟣 **T Pequeno** — topo + haste 1
- 🔵 **T Médio** — topo + haste 2
- 🟢 **T Grande** — topo + haste 3

## Controlos
| Ação | Teclado | Touch |
|------|---------|-------|
| Mover | ← → | ◀ ▶ |
| Descer | ↓ | ▼ |
| Rodar | ↑ | RODAR |
| Hard drop | SPACE | DROP |
| Pausa | P | PAUSA |

## Mensagens
- ✅ **COMPILADO!** — linha(s) limpa(s)
- ❌ **FALHA CRÍTICA! STACK OVERFLOW!** — game over
